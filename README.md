# 📊 Interactive Sorting Algorithm Visualizer

An interactive web application that demonstrates how different sorting algorithms work through real-time animations. This project helps users understand the step-by-step execution of sorting algorithms in a visual and engaging way.

## 🚀 Features

- Generate a random array
- Bubble Sort visualization
- Merge Sort visualization
- Real-time sorting animations
- Interactive and user-friendly interface
- Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
Interactive-Sorting-Algorithm-Visualizer/
│── index.html
│── style.css
│── script.js
```

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/pranjali123-coder/Interactive-Sorting-Algorithm-Visualizer.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.
